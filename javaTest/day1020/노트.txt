인터페이스
- 클래스가 자료형인것 처럼 인터페이스도 자료형이다
- 완전 추상의 개념
- 추상메소드와 상수만 가질수 있다
- 메소드에는 public abstract 키워드가 생략되어있다.
- 상수에는 public final이 생략
- 인터페이스 구현할 때 implements 사용

class abstract A {
	public abstract void pro();
}

class B extends A {
	public void pro() {}
}
===============================
interface A {
	public abstract void pro();
}

class B implements A {
	public void pro() {};
}
