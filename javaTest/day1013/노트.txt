오류의 종류
- 문법오류
	컴파일 x
	자바 문법에 맞지 않는 오류
- 논리오류
	컴파일도 되고 실행도 되지만 의도한데로 동작하지 않는다
	
- 실행오류
	컴파일이 되지만 실행이 될떄와 안될떄가 있다
	(사용자의 부주의로 실행중 발생하는 오류)
	
메소드 :
- 어떤 문제해결을 하기위한 (어떤 기능을 수행하기 위한)
  서로 관련있는 명령어들의 집합
- 어떤한 기능이 여러차례 필요할 때 그 기능을 모아서 메소드를 만들어 두면
  효율적으로 사용가능
- 처리해야할 기능들이 많을 떄 하나의 메인 메소드에 모두 처리하게 되면
  메인 메소드가 복잡해진다
- 기능별로 따르 메소드를 만들어 분리해 두면 코드가 간결해진다
- 메소드의 역할에 떄라 전달해야 하는 값(매개변수)이 있을 수 있고 없을 수 있다

메소드 형식
public 반환자료형 메소드이름([매개변수]...) {
	명령어;
	return;
}
- 메소드의 역할에 따라 매개변수가 있거나 없다
- 메소드의 역할에 따라 반환값이 있거나 없다
- 반환값이 없을 떄는 반환자료형에 void
- 반환값이 있을 때는 반환자료형에 반환값의 자료형을 쓴다
