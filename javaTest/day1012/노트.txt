메소드
- 어떤 기능을 수행하기 위한 명령어들의 모임
- 메소드는 호출문에 의해서 동작
- 메소드의 body가 끝나면 메소드 호출하는 곳으로 return
- 이때 메소드에 따라 return값이 있을 수도 없을 수도 있다

public - 공용(누구나 쓸 수 있다)

public void 메소드명(데이터 타입) {
	명령어
}
- void일 떄는 return 사용 불가
- return값이 없으므로 출력문에 사용 불가능

public 자료형 메소드명(매개변수) {
	명령어
	retunr 값
}
- return할 자료형에 따라 변한다

오버로딩
- 같은 이름의 메소드를 중복 정의하는 것
메소드를 만들때 같은 이름 메소드를 만든다면
매개변수의 개수가 다르거나 자료형이 달라야한다
